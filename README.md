# Case-Sharepoint-STS-Cidade-Tiradentes
Este projeto consistiu no desenvolvimento e implementação de uma plataforma no SharePoint para a Supervisão Técnica de Saúde da Cidade Tiradentes. O objetivo foi centralizar informações, garantir a segurança de arquivos e otimizar a comunicação entre equipes de diferentes turnos. Conta com Cultura de Dados e Troubleshooting, Acesso multiplataforma.
